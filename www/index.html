CSSANIMATE IS STYLISH AND CUTE
  •  Translate X
    $('.box').cssanimate({ x: 30 })
  •  Translate Y
    $('.box').cssanimate({ y: 30 })
  •  Translate X / Y
    $('.box').cssanimate({ x: 30, y: 30 })
  •  Rotate X
    $('.box').cssanimate({ rotatex: 180 })
  •  Rotate Y
    $('.box').cssanimate({ rotatey: 180 })
  •  Rotate X / Y
    $('.box').cssanimate({ rotate: [1, 1, 0, 180] })
  •  Scale X
    $('.box').cssanimate({ scalex: 4 })
  •  Scale Y
    $('.box').cssanimate({ scaley: 4 })
  •  Scale X / Y
    $('.box').cssanimate({ scalex: 4, scaley: 4 })
  •  Skew X
    $('.box').cssanimate({ skewx: 45 })
  •  Skew Y
    $('.box').cssanimate({ skewy: 45 })
  •  Skew X / Y
    $('.box').cssanimate({ skewx: 45, skewy: 45 })
  •  Opacity
    $('.box').cssanimate({ opacity: 0 })
  •  Width
    $('.box').cssanimate({ width: 120 })
  •  Radius
    $('.box').cssanimate({ border-radius: 30 })

  • method chain

    method chain is supported. Animation will be executed serially exactly like animate of jQuery.
    e.g. $('.box') .cssanimate({ x: 100, y: 0, scale: 0.5, rotatez: 90 }) .cssanimate({ x: 100, y: 100, scale: 1, rotatez: 180 }) .cssanimate({ x: -100, y: 100, scale: 0.5, rotatez: 270 }) .cssanimate({ x: -100, y: 0, scale: 1, rotatez: 360 }) .cssanimate({ x: 0, y: 0, scale: 1 })

  • origin

    You can use origin to set where rotations and scales start from.
    e.g. $('.box') .origin(0, 0) .cssanimate({ rotatez: 45 }) alternate origin('50%', '50%') origin('bottom', 'right') origin('30px', '20px')
