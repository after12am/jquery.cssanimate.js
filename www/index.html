jquery.cssanimate.js is a jQuery plugin that provides smooth animation for smartphone. This plugin uses css3 in core. Using css3 would give your app lightness and interactivity especially in mobile for using gpu in contrast to Javascript. But css3 is a bit difficult to controll under dynamic usage. So this plugin designed to take advantage of knowledge of jquery of you.
