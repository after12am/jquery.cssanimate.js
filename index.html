method chain

  Method chain is supported. Animation will be executed serially exactly like animate of jQuery.
e.g. 
$('.box')
    .tween({ x: 100,  y: 0,   scale: 0.5, rotatez: 90  })
    .tween({ x: 100,  y: 100, scale: 1,   rotatez: 180 })
    .tween({ x: -100, y: 100, scale: 0.5, rotatez: 270 })
    .tween({ x: -100, y: 0,   scale: 1,   rotatez: 360 })
    .tween({ x: 0,    y: 0,   scale: 1,   rotatez: 0   })

perspective

  You can control elements using 3D transforms. Set perspective value which is how 3D elements are viewed. If want to disable 3d transformation, set false.
e.g. 
$('.box').tween({
    perspective: 100, // false
    x: 50,
    z: 50,
    rotatey: 60
});

Transform Any CSS

  You can transform any css properties. The example shows how to change the color and width.
e.g. 
$('.box').tween({
    width: 0,
    backgroundColor: '#CF3932'
});

Prefix Free

  Prefix free that free you from vendor prefix hell is supported. You can use unprefixed css property everywhere.
e.g. 
$('.box').tween({
    transform: 'translateX(48px)'
});

Relative Value

  Relative value allows you to start with '+=' or '-='.
e.g. 
$('.box').tween({
    perspective: 100,
    z: '-=100'
});

$.fn.css

  $.fn.css helps you to get the value of a style property for the first element in the set of matched elements and helps you to set css properties for every matched element.
e.g. 
$(this)
    .css('rotatez', 45)
    .text($(this).css('rotatez'));

ease

  You can use advanced easing function. 29 types are supported.
e.g. 
$('.box).tween({
    scaleX: 3,
    rotate: 360,
    backgroundColor: '#CF3932',
    ease: 'snap'
});
